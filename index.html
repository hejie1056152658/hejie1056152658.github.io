程序代码

数组几种转换

reverse() //颠倒数组中元素的顺序。
pop() //删除数组的最后一个元素 并原数组的长度也会改变
concat() //连接两个或更多的数组，并返回结果。
join() //把数组的所有元素放入一个字符串。元素通过指定的分隔符进行分隔。
push() //向数组的末尾添加一个或更多元素，并返回新的长度。

shift() //删除并返回数组的第一个元素
slice() //从某个已有的数组返回选定的元素
sort() //对数组的元素进行排序
splice() //删除元素，并向数组添加新元素。
toSource() //返回该对象的源代码。
toString() //把数组转换为字符串，并返回结果。
toLocaleString() //把数组转换为本地数组，并返回结果。
unshift() //向数组的开头添加一个或更多元素，并返回新的长度。
valueOf() //返回数组对象的原始值

shall命令

git学习日志 

## 基本命令

q 退出浏览
翻页使用空格
tab 智能补全
echo 打印内容
git init  将当前所在的文件夹 用git版本控制工具进行管理  在当前文件夹中生成一个。git隐藏文件夹 该文件夹就是git用来存取每次修改的历史记录
ls -a 显示所有文件下的内容
git status  查看当前文件夹的状态 如果文件没有被git管理 会有untracked files提示
git add  用来将文件添加到git中 让git进行管理
git add 文件名 文件名

git add *  将所有的文件全部添加到git中管理
git add -A
git commit -m "提示信息"   将当前的文件状态保存一份到版本库中 生成一个新版本 历史记录会有这个版本的记录
git log 用来查看所有已经保存的版本的内容
git reset --hard 版本号   将代码恢复到指定的版本
pwd    显示在哪个位置
touch  创建文件  +文件名  或 +文件路径
cd     改变当前所处的目录
mkdir  创建文件夹    +文件名  或 +文件路径
clear  清屏
rm     删除指定的文件    +文件名  或 +文件路径
rm（rmdir）  删除文件夹（只能删除空文件夹）
rm -r  删除有内容的文件夹 （-r指递归的意思 递归删除  先删里边东西  再删文件夹 注意安全删除）
rm -f   强制删除
cp 复制文件    格式：cp 源文件（名称加路径）+ 目标文件（名称加路径）
cp -r +源文件夹+目标文件夹   可以用来复制文件夹
mv +源文件+目标文件  移动文件
mv 执行重命名可
cat 展示文价内容   +文件名  或 +文件路径  是将所有内容一次性展示
less 分页展示文件内容  +文件名  或 +文件路径


## 分支命令

git branch + 分支名称       创建分支！
git branch                  查看所有分支
git checkout + 分支名称     切换到分支
git merge    + 分支名称     合并分支
git branch -d + 分支名称    删除分支
git checkout -b +分支名称   快速创建并切换分支


## 远程仓库

git init --bare                                  创建远程仓库
git push + 远程仓库地址 master                   向远程仓库推送内容
git clone + 远程仓库地址 + 本地文件夹名称        从远程仓库获取内容
git pull  + 远程仓库地址 + 分支名称              从远程仓库获取更新的内容     一般在push之前都会先pull一下
